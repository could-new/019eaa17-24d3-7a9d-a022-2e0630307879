# Niche Finder App (Поиск прибыльных ниш)

Приложение для поиска и аналитики прибыльных ниш на популярных маркетплейсах (Wildberries и Ozon). Приложение предоставляет инструмент для селлеров, позволяющий оценивать выручку, конкуренцию и средний чек в различных категориях.

## Функционал
- Выбор маркетплейса (Wildberries или Ozon).
- Поиск и фильтрация ниш по категориям.
- Просмотр списка ниш с краткими показателями.
- Детальная статистика по каждой нише (выручка, количество конкурентов, средний чек, тренды).

## Стек технологий
- Flutter
- Dart

## Запуск проекта
1. Убедитесь, что у вас установлен Flutter SDK.
2. Выполните команду `flutter pub get` для установки зависимостей.
3. Запустите проект командой `flutter run`.

## Об архитектуре
В приложении используется простая архитектура с моковыми данными для демонстрации функционала. В будущем можно подключить реальные API для получения аналитики в реальном времени.
